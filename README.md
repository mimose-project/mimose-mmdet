## Require

- V100
- CUDA 11.3, cudnn 8.3
- Python 3.8
- PyTorch 1.11, mmcv-full 1.5.0 +

## Install 

1. ```git clone git@github.com:open-mmlab/mmcv.git```, and checkout to the tag ```v1.5.2```
2. ```pip install ./```
